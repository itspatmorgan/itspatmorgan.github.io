Task tracking for the portfolio site lives on the [Personal Website](https://github.com/users/itspatmorgan/projects/2) project board and [GitHub Issues](https://github.com/itspatmorgan/itspatmorgan.github.io/issues).

## How we work

This site is built collaboratively with [Claude Code](https://claude.ai/code). Issues, commits, and PR descriptions document the planning, decisions, and implementation process as it happens.

**Labels** describe two dimensions of each task:

| Label | Description |
|-------|-------------|
| `home` | Home page |
| `resume` | Resume page |
| `projects` | Project detail pages |
| `writing` | Writing section |
| `design` | Layout, styling, typography, branding, animations |
| `content` | Articles, case studies, media assets |
| `infrastructure` | Build, deploy, tooling, dev workflow |
| `documentation` | Docs, guides, READMEs |
